I have made some changes to the SRFI 89 document with the goal of finalizing the SRFI soon. Please send your comments ASAP. 

Summary of changes:

1) Changed name of SRFI to: "Optional positional and named parameters".

2) Removed requirement that "[" and "]" be synonymous to "(" and ")".
3) Removed requirement that "define" and "lambda" be extended. The new forms "define*" and "lambda*" have been added.
4) Updated grammar, examples and reference implementation to be consistent with the above changes. 

5) Updated HTML links to other SRFIs.
