Although only a few existing Schemes provide the whole environment as
an a-list, I believe that SRFI-98 should at least standardize this as an
optional procedure if not a required one.  It's true that ANSI C does not
provide access to the whole environment, but both Posix and Windows do,
as well as pre-Posix Unix-like systems.
