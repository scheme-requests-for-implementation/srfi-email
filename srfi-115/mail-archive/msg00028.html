Why are w/ascii and w/unicode necessary? The ascii character set can be used instead. 

(regexp-search `(: bos (* ,char-set:ascii) eos) "English") => #<rx-match>
(regexp-search `(: bos (* ,char-set:ascii) eos) "Ελληνική") => #f
