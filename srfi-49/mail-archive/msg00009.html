Hm, note that the formatter I posted a few seconds ago does not
reformat S-expressions, but Scheme values (lists, etc) as
I-expressions - so a full solution to Svens problem would include a
main progam that loops over a file, repeatedly using (iformat ""
(read)) to read in its expressions, and format them, and writes the
results to a new file.
