I would like to add that I think J.D. Stone's "section" seems to
me the most appropriate name, since this form is really like an 
operator section in Haskell:

  (+ 1)        <==>       \x -> x + 1        <==>    (section + <> 1)
  (1.0 /)      <==>       \x -> 1.0 / x      <==>    (section / 1.0 <>)
