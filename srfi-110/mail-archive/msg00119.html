On Sun, Apr 14, 2013 at 3:29 PM, David A. Wheeler <dwheeler@xxxxxxxxxxxx> wrote:
Attached is an initial reference implementation of:
* curly-write-simple and neoteric-write-simple
* [curly,neoteric]-write-[shared,cyclic]
It then does:
Â(define neoteric-write neoteric-write-cyclic)
Â(define curly-write curly-write-cyclic)

Comments/improvements welcome! ÂThe "simple" implementations is separable from the shared/cyclic implementation, so that those who don't want shared and cyclic versions can omit them.

I used the Chibi Scheme implementation as a starting point. ÂThat builds on srfi-38, but it uses hash tables (more efficient) and can deal with both cyclic and shared structure versions. ÂThe Chibi code for this is released to the public domain, so no license issue.

Below is the starting test set and its output, to show that it's plausible. ÂThe test set is used by all; I can't (obviously) include cycles in the calls to the -simple variants.
